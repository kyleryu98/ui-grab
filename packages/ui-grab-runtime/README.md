# Internal Runtime

This package contains the internal runtime source for the `ui-grab` public package.

- Public install instructions live in the repository root [README](https://github.com/Yongtaek-Ryu/ui-grab#readme).
- End users should install `ui-grab`, not this private workspace package.
- The runtime keeps the original prompt UI structure and adds built-in `Shift + click` multi-select support.
