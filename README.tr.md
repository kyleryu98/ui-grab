### Kurulum

```bash
npx ui-grab@latest init
pnpm add -D ui-grab

pnpm install
pnpm build
pnpm --filter ui-grab pack
```

> [!NOTE]
> `ui-grab`, bu forkun herkese açık paket adıdır. Depo yayıma hazırdır, ancak ilk herkese açık npm sürümü hâlâ bakımcı hesabının npm doğrulamasını gerektirir.

### Neden UI Grab

- Dosya, bileşen ve HTML bağlamını doğrudan tarayıcıdan kopyalayın
- Orijinal React Grab prompt arayüzünü ve yorum geçmişini koruyun
- Yorum modunda yerleşik `Shift + tıkla` çoklu seçimi kullanın
- Birden fazla öğe gönderildikten sonra gruplanmış prompt geçmişini koruyun
- `ui-grab`, `ui-grab/core`, `ui-grab/primitives` ve `ui-grab/styles.css` herkese açık giriş noktalarını koruyun

### Kullanım

1. Yüzen araç çubuğundan seçim aracını etkinleştirin.
2. Bir öğenin üzerine gelin ve tek bir öğeyi kopyalamak için `Cmd+C` veya `Ctrl+C` tuşlarına basın.
3. Gruplu seçim oluşturmak için `Shift` tuşunu basılı tutup birden fazla öğeye tıklayın.
4. `Shift` tuşunu bırakın, orijinal textarea'ya yazın ve gönderin.

### Manuel Kurulum

#### Next.js (App Router)

```tsx
import Script from "next/script";

export default function RootLayout(props: { children: React.ReactNode }) {
  return (
    <html>
      <head>
        {process.env.NODE_ENV === "development" && (
          <Script
            src="//unpkg.com/ui-grab/dist/index.global.js"
            crossOrigin="anonymous"
            strategy="beforeInteractive"
          />
        )}
      </head>
      <body>{props.children}</body>
    </html>
  );
}
```

#### Vite

```tsx
if (import.meta.env.DEV) {
  import("ui-grab");
}
```
